Permissionless Settlement

TxPools uses automatic keeper-based settlement for smooth UX, but the settlement instruction is permissionless.

Anyone can trigger settlement after a match ends. The Solana program verifies the final result through TxLINE before marking the pool as settled. Once settled, winners claim their proportional USDC payout.
